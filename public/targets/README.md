# 목표 이미지 (타수별)

이 폴더에 **만들어야 할 화면**을 `image_{n}` 형식으로 넣으세요.

- `n` = **타수(stroke)**. 홀은 1개로 고정이고, 타수마다 해당 이미지가 "만들어야 할 화면"으로 표시됩니다.
- 예: `image_1.png`, `image_2.png`, `image_3.png` …
- 지원 확장자: `png`, `jpg`, `jpeg`, `webp`, `gif`, `svg`
  - ⚠️ Gemini 비전 비교는 **png / jpg / webp** 를 권장합니다. (현재 들어있는 `.svg` 3개는 동작 확인용 샘플이며, 자유롭게 교체/삭제하세요.)
- 권장 해상도: **1024 × 640** (캡처 비율과 동일).

이미지 개수가 곧 플레이 가능한 **타수**가 됩니다. 마지막 이미지까지 스윙하면 홀 종료입니다.

게임은 시작 시 `/api/targets`로 이 폴더를 읽어 `image_{n}`을 번호순으로 정렬해 사용합니다.
